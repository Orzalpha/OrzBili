# OrzBili

## 简介

这是个乱七八糟的 Bilibili 客户端项目，是初学者 Orzalpha 不懂事写着玩的。

这家伙准备完成以下几件事：

* [x] 登陆，并能保存登陆状态
* [x]  查看追番列表
* [x]  番剧播放
* [ ]  神秘的其他功能

让我们衷心地祈祷这项目能够顺利完工，而不是半路就流产了。

以及，如果上述提到的这些功能真的能够实现的话，某杖可能还会追加一些其他功能：

* 弹幕
* 下载
